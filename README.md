## 关于MCP

Model Context Protocol (MCP) 是一个开放协议，用于标准化应用程序如何向大型语言模型(LLM)提供上下文。MCP就像AI应用程序的USB-C接口，提供了一种标准化的方式来连接AI模型与不同的数据源和工具。
